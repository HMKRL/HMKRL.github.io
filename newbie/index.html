我如果想在Linux玩Game或安裝軟體
聽起來好像很厲害齁！在有APT套件管理的Linux就只要一行
  • sudo apt-get install 要裝的套件
指令前面會有sudo是為了讓我們用root權限來執行這個指令
要解除安裝但不包括設定檔的話
  • sudo apt-get remove 要刪的套件
要完全移除且包括設定檔的話
  • sudo apt-get purge 要刪的套件
  • 用remove並不能完全移除設定檔，若要完全移除請採用sudo apt-get purge
  • 那麼之前用remove或autoremove沒purge掉的軟體怎麼辦呢，因為只有remove而沒有被purge的套件會有^rc標記，那就可以用dpkg的指令與apt-get的組合：
  • sudo apt-get purge `dpkg -l | grep ^rc | awk '{ print $2 }'`
  • 就可以完整移除了
如果用的是採取RPM管理的Linux，那麼也跟上面類似喔
安裝：
  • sudo yum install 要裝的套件
移除：
  • sudo yum remove 要刪的套件
用指令就可以完成安裝移除與更新囉

  1. 打im-config，會跳出視窗。按"確定"或"yes"，一直到下面這個視窗再選hime
